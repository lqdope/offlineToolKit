# 🛠️ offlineToolBox Pro

Современный, ультра-легковесный и минималистичный цифровой мультитул в формате «все в одном». Набор утилит для повседневных задач разработчика и продуктивной работы, способный функционировать полностью локально и автономно прямо в вашем браузере.

Проект спроектирован с акцентом на приватность, высокое быстродействие и чистый UI в стиле модульного минимализма.

P.S старайтесь сохранить текущий стек проекта(при форках и улучшениях)

## 🚀 Основной функционал

### 🤖 1. ИИ-Помощник (AI Workspace)
* **Локальный Чат с LLM:** Встроенный веб-интерфейс для прямого взаимодействия с локальной нейросетью.
* **Гибкие настройки подключения:** Возможность «на лету» изменить URL-хост (Endpoint) и имя модели Ollama прямо из интерфейса.
* **Голосовой ввод (Speech-to-Text):** Нативная поддержка распознавания русской речи через Web Speech API — надиктуйте промпт вместо ввода с клавиатуры.
* **Мультимодальность (Vision):** Поддержка прикрепления изображений (автоматическая конвертация в Base64) для анализа локальными Vision-моделями (например, `llava`).

### 🔐 2. Безопасность и Пароли (Security Suite)
* **Продвинутый генератор паролей:** Кастомизация длины, регистра, цифр и спецсимволов + быстрое копирование в буфер обмена.
* **Встроенный менеджер паролей:** Мгновенное сохранение сгенерированных ключей с добавлением ярлыков во внутреннюю изолированную базу браузера (`localStorage`).

### 📥 3. Сеть, Загрузки и Парсинг (Network & Scraper)
* **Умный Downloader:** Прямое скачивание файлов через браузерные стримы (`Blob`), предотвращающее автооткрытие медиафайлов во вкладках.
* **Парсер картинок (HTML Scraper):** Извлечение ссылок на изображения из вставленного сырого HTML-кода страницы без лишних бэкенд-запросов.

### 🔄 4. Конвертеры и Инструменты Разработчика (DevTools)
* **Оптимизатор в WebP:** Сжатие и конвертация изображений (PNG/JPG -> WebP) на клиенте через HTML5 Canvas в один клик.
* **Форматтер JSON:** Превращение минифицированного или «грязного» JSON в структурированный читаемый вид с валидацией синтаксиса.
* **Анализатор текста:** Подсчет количества символов, слов и хронометража чтения в реальном времени.

## 🎨 Особенности интерфейса

* **Модульная сетка (CSS Grid):** Каждый инструмент представляет собой изолированную контурную карточку. Интерфейс динамически адаптируется под любые экраны.
* **Нативный Dark Mode:** Тема оформления автоматически переключается на темную или светлую, подстраиваясь под настройки вашей операционной системы.
* **Zero Dependencies:** Приложение написано на чистом HTML5, CSS3 и Vanilla JS. Никакого `npm install`, Webpack или сторонних тяжелых библиотек.

## ⚙️ Настройка локального ИИ (Ollama)

Чтобы использовать встроенный ИИ-чат, запустите Ollama на вашем компьютере, разрешив CORS-запросы (иначе браузер заблокирует отправку промптов):

* **Windows (в PowerShell перед запуском Ollama):**
  ```bash
  $env:OLLAMA_ORIGINS="*"
  ollama serve
* **macOS / Linux (в Терминале):**
  ```bash
  OLLAMA_ORIGINS="*" ollama serve
